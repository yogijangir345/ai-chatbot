# AI Chatbot Backend Project

## 📌 Project Overview
This is a simple AI Chatbot backend project made using Node.js, Express, and CORS.  
The main idea of this project is to connect a backend server with an AI API so users can send messages and get AI-generated replies.

## ⚙️ Technologies Used
- Node.js
- Express.js
- CORS
- JSON API handling

## 🚀 How to Run the Project
1. Install dependencies
2. Start the server
3. Server will run on

## 📡 API Setup
This project uses an AI API key for generating responses.

At first, I tried to complete the full project using the API key, but I faced some issues like:

- API quota limit error
- Request limit exceeded problem

Because of these issues, the AI responses were not working properly at the time of development.

## ⚠️ Project Limitation
Due to API key quota/error issues and project deadline pressure, I was not able to fully complete the AI integration part.

So I decided to submit a basic working backend structure instead of a full AI response system.
